# Passage Motion Demos

Interactive UI card demos with Framer Motion animations.

## Demos

- **Jackie Demo** (`/jackie-demo`) - AI Counsellor chat interface with animated conversation flow
- **Training Demo** (`/training-demo`) - Admissions portal with student management

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 16
- React
- Tailwind CSS
- Framer Motion
